# QwenPaw 产品功能展示图片生成 Prompts

本文档包含10个不同场景的QwenPaw产品功能展示图片生成提示词，每个prompt都基于QwenPaw的真实功能特性，并参考GPT Image 2.0的最佳实践进行优化。

**适用工具**：ChatGPT Image 2.0（推荐）、DALL-E 3、Midjourney、Stable Diffusion

**核心优化**：
- ✅ 采用"版式+主体+场景+风格+光线+镜头"的黄金公式
- ✅ 明确指定宽高比（9:16、16:9、1:1等）
- ✅ 中文文字用引号标出，避免AI自行编造
- ✅ 具体风格参考（如"柯达Portra 400"、"吉卜力风格"）
- ✅ 强调留白与层级，避免画面杂乱

### 5. Prompt调整技巧（基于GPT Image 2.0最佳实践）

#### 5.1 黄金公式应用
所有优化版prompt都遵循这个公式：
```
版式 + 主体 + 场景 + 风格 + 光线 + 镜头 + 文字约束 + 用途
```

#### 5.2 常见调整点

**数据个性化**：
- 将"23次收获"改为实际数据
- 将"95.6%成功率"改为真实统计
- 将"2小时前"改为实际时间

**配色调整**：
- 统一品牌色：将所有"#FF7F16"替换为你的品牌色
- 调整情感色调：温暖色系（橙红黄）vs 冷静色系（蓝绿紫）

**风格微调**：
- 添加"dark mode"切换为暗色主题
- 添加"glassmorphism"启用玻璃态效果
- 添加"neumorphism"启用新拟态风格
- 添加"柯达Portra 400"增加胶片感

**文字控制**（关键）：
- 所有需要精确显示的中文必须用引号标注
- 同时指定字体："宋体"、"楷体"、"黑体"、"手写体"
- 添加"文字清晰无错乱"强调文字质量

**镜头控制**：
- "50mm f/2.8"：自然人像视角，浅景深
- "24mm f/8"：广角风景，深景深
- "85mm f/1.4"：特写肖像，极浅景深
- "中画幅相机"：更高画质和自然虚化

**迭代优化策略**：
1. 第一次：使用完整prompt生成基础版
2. 第二次：用"在此基础上，把XX改成YY"触发局部修改
3. 第三次：继续细化细节，不要推倒重来

## 快速参考：GPT Image 2.0使用技巧

### ChatGPT Image 2.0 新特性（2026年4月）
1. **中文渲染完美**：字级准确率接近99%
2. **Thinking模式**：Plus/Pro用户可开启，生图前AI会思考
3. **照片级真实感**：光影、纹理、结构高度还原
4. **精准局部编辑**：可对画面特定区域修改
5. **高分辨率**：最高4096×4096，支持1:3至3:1极端比例

### 快速上手步骤
```
1. 登录 https://chatgpt.com
2. 点击输入框左侧「+」→ 选择「创建图片」
3. 粘贴本文档的优化版Prompt
4. （Plus/Pro用户）开启Thinking模式效果更好
5. 生成后可点击图片进行局部编辑或调整比例
```

### Prompt编写核心技巧
```
✅ DO（推荐做法）
- 先说版式（16:9、9:16、1:1）
- 用具体风格词（柯达Portra 400、吉卜力风格）
- 中文文字用引号标出："显示的文字内容"
- 声明留白："留白合理、信息主次清晰"
- 用镜头参数控制景深："50mm f/2.8"
- 分步迭代："在此基础上，把XX改成YY"

❌ DON'T（避免做法）
- 模糊描述："好看"、"高级"、"有质感"
- 让AI自由发挥文字内容
- 一次性描述过多细节（超过3个场景）
- 生图不满意就完全重写prompt（应该迭代优化）
```

### 常见问题排查
| 问题 | 原因 | 解决方案 |
|-----|------|---------|
| 中文文字错乱 | AI自行编造 | 用引号标注："要显示的内容" |
| 画面过于拥挤 | 未声明留白 | 加入"留白合理、主次清晰" |
| 风格不够明确 | 描述太抽象 | 用具体参考"柯达Portra 400胶片" |
| 景深不自然 | 未指定镜头 | 添加"50mm f/2.8"等参数 |
| 局部不满意 | 未迭代优化 | 用"在此基础上改XX"而非重新生成 |
